# ExamGrader - 智能试卷批改系统

## 项目概述

ExamGrader是一个基于OPEA组件的智能教育解决方案，实现了试卷自动批改、错题智能分析和个性化复习建议的完整学习闭环。

## 痛点分析

1. **教师负担重** - 大量试卷批改耗时费力
2. **反馈不及时** - 学生等待批改结果时间长
3. **错题无追踪** - 学生难以系统复习错题
4. **复习无方向** - 缺乏针对性的复习建议

## 解决方案

### 核心功能

1. **智能批改引擎**
   - 支持选择题、填空题、问答题等多种题型
   - AI驱动的主观题评分
   - 自动化批改流程

2. **错题管理系统**
   - 自动收集学生错题
   - 建立个人错题本
   - 知识点关联分析

3. **个性化学习推荐**
   - 基于错题分析薄弱知识点
   - 生成针对性复习建议
   - 一学生一卷的检查卷生成

## 技术亮点

1. **OPEA原生集成** - 完全基于OPEA组件构建
2. **向量数据库** - pgvector实现智能检索
3. **流式响应** - 实时批改进度反馈
4. **个性化推荐** - 基于知识点的智能分析

## 演示流程

1. 上传试卷和参考答案
2. 上传学生答卷
3. 启动自动批改
4. 查看批改结果和统计
5. 查看错题分析和复习建议
6. 生成个性化检查卷
