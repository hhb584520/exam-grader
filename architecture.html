  • еӯҰз”ҹзӯ”жЎҲжҸҗдәӨ
  • жӯҘйӘӨеҲҶжһҗжҺҘеҸЈ
  • еӨҚд№ е»әи®®жҺҘеҸЈ
  • жЈҖжҹҘеҚ·з”ҹжҲҗжҺҘеҸЈ
рҹӨ– Agent Service
еҹәдәҺOPEA Compsзҡ„жҷәиғҪд»ЈзҗҶжңҚеҠЎ

  • жӯҘйӘӨзә§й”ҷиҜҜеҲҶжһҗ
  • RAGзҹҘиҜҶеә“жҹҘиҜў
  • дјҳеҢ–ж–№жі•жҺЁиҚҗ
  • дёӘжҖ§еҢ–еӯҰд№ е»әи®®
рҹ’ҫ Database Service

  • иҜ•еҚ·ж•°жҚ®еӯҳеӮЁ
  • й”ҷйўҳи®°еҪ•з®ЎзҗҶ
  • зҹҘиҜҶзӮ№зҹҘиҜҶеә“
  • жӯҘйӘӨжЁЎжқҝеӯҳеӮЁ
  • еҗ‘йҮҸзӣёдјјеәҰжЈҖзҙў

еҹәдәҺvLLMзҡ„еӨ§иҜӯиЁҖжЁЎеһӢжңҚеҠЎ

  • жӯҘйӘӨеҲҶжһҗз”ҹжҲҗ
  • ж•ҷеӯҰе»әи®®з”ҹжҲҗ
  • жЈҖжҹҘеҚ·иҮӘеҠЁз”ҹжҲҗ

6. з”ҹжҲҗжЈҖжҹҘеҚ· вҶ’ еүҚз«ҜQuizGenerator вҶ’ API Service вҶ’ Agent Service вҶ’ LLMж №жҚ®й”ҷйўҳз”ҹжҲҗжЈҖжҹҘеҚ·

рҹ“Ұ еҠҹиғҪжЁЎеқ—жһ¶жһ„

рҹ“қ

иҜ•еҚ·жү№ж”№жЁЎеқ—

  • иҜ•еҚ·дёҠдј дёҺи§Јжһҗ
  • еӯҰз”ҹзӯ”жЎҲжҺҘж”¶
  • AIжҷәиғҪжү№ж”№
  • жҲҗз»©и®Ўз®—
рҹ”Қ

жӯҘйӘӨеҲҶжһҗжЁЎеқ—

  • и§ЈйўҳжӯҘйӘӨеҲҶи§Ј
  • й”ҷиҜҜе®ҡдҪҚеҲҶжһҗ
  • RAGзҹҘиҜҶеә“жЈҖзҙў
  • дјҳеҢ–ж–№жі•жҺЁиҚҗ
рҹ“ҡ

й”ҷйўҳз®ЎзҗҶжЁЎеқ—

  • й”ҷйўҳиҮӘеҠЁж”¶йӣҶ
  • зҹҘиҜҶзӮ№е…іиҒ”
  • й”ҷйўҳжң¬з®ЎзҗҶ
  • еӨҚд№ иҝӣеәҰи·ҹиёӘ
рҹҺҜ

еӨҚд№ е»әи®®жЁЎеқ—

  • и–„ејұзҹҘиҜҶзӮ№еҲҶжһҗ
  • дёӘжҖ§еҢ–е»әи®®з”ҹжҲҗ
  • еӨҚд№ и®ЎеҲ’еҲ¶е®ҡ
  • иө„жәҗжҺЁиҚҗ
рҹ“Ӣ

жЈҖжҹҘеҚ·жЁЎеқ—

  • дёӘжҖ§еҢ–йўҳзӣ®з”ҹжҲҗ
  • йҡҫеәҰиҮӘйҖӮеә”
  • зҹҘиҜҶзӮ№иҰҶзӣ–
  • зӯ”жЎҲдёҺи§Јжһҗз”ҹжҲҗ
рҹ‘ЁвҖҚрҹҸ«

ж•ҷеӯҰж”ҜжҢҒжЁЎеқ—

  • еӯҰз”ҹеӯҰд№ з”»еғҸ
  • зҸӯзә§еҲҶжһҗжҠҘе‘Ҡ
  • ж•ҷеӯҰе»әи®®з”ҹжҲҗ
  • еӯҰд№ ж•°жҚ®еҲҶжһҗ
