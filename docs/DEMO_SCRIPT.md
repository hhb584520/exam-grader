## Recording Guide

### Preparation
1. **Environment Setup**
   - Start LLM Service (vLLM)
   - Start Embedding Service
   - Start PostgreSQL + pgvector
   - Start Agent Service
   - Start API Service
   - Start Web UI (http://localhost:5173)

2. **Test Data Preparation**
   - Prepare sample exam papers (multiple choice, fill-in, essay questions)
   - Prepare student answer sheets
   - Ensure test data exists in database

3. **Recording Tools**
   - Screen recording software (OBS Studio, Camtasia)
   - Quality microphone for clear audio
   - Prepare presenter narration

### Recording Steps
1. Open browser, navigate to http://localhost:5173
2. Home page is displayed by default - begin with introduction
3. Click "Get Started" to upload a test exam
4. Demonstrate features in script order
5. Ensure smooth transitions between sections
6. Record without narration first, add voiceover later
7. Verify all features work correctly

### Post-Production
1. Add soft background music
2. Add professional narration
3. Add synchronized subtitles
4. Add smooth transition effects
5. Adjust video pacing
6. Export as MP4 format

### Exam Content

```
Exam ID: math_test_001
Title: High School Math Midterm Exam
Subject: Mathematics

1. Multiple Choice (5 points each)
   1.1 Given function f(x) = x² + 2x + 1, then f(2) = ?
       A. 5 B. 8 C. 9 D. 10
   
2. Fill-in-the-blank (5 points each)
   2.1 The 10th term of arithmetic sequence 1, 3, 5, 7, ... is ____

3. Essay Questions (20 points each)
   3.1 Find the extreme points of function y = x³ - 3x² + 2
       and determine whether they are maximum or minimum
```

### Student Answer Example
```json
{
  "q1": "C",
  "q2": "18",
  "q3": "Derive y'=3x²-6x"
}
```

### Sample Problem for Step Analysis
```
Problem: Find extreme points of f(x) = x³ - 3x² + 2 and determine max/min

Student's Solution:
1. f'(x) = 3x² - 6x
2. Set f'(x) = 0: 3x² - 6x = 0
3. Solve: x = 0
4. Therefore x=0 is extreme point

Errors: 
- Step 3: Missing x = 2 solution
- Step 4: Missing second derivative analysis
```
